# WAP 画布选择器底部抽屉化设计

## 目标

将 WAP 生图页中的画布比例选择器从原生下拉改为更适合移动端点按的底部抽屉列表，在控制内容区高度的同时提升比例识别效率与点按命中率。

## 设计结论

采用底部抽屉加单列卡片列表的方式展示画布比例选项。

当前方案包含以下要点：

1. 页面参数区保留一行触发器，用于展示当前所选比例摘要。
2. 点击触发器后，从底部弹出抽屉。
3. 抽屉内使用单列卡片列表展示全部比例选项。
4. 每个卡片项显示比例缩略预览、比例名称、场景说明。
5. 选中任一项后立即关闭抽屉，并更新当前摘要。

## 页面结构

### 触发器

触发器位于原“画布比例”区域，保持紧凑高度。

展示内容包括：

1. 左侧比例缩略预览
2. 中间主标题：`比例 + 名称`，例如 `16:9 宽屏`
3. 次级说明：场景说明，例如 `电影宽屏`
4. 右侧展开箭头

这样可以在未展开时提供足够的当前状态信息，同时避免原 10 项按钮网格造成的纵向挤压。

### 底部抽屉

抽屉从底部弹出，适合移动端拇指操作。

约束如下：

1. 标题为 `选择画布比例`
2. 顶部加入拖拽指示条样式，强化抽屉感知
3. 最大高度约为视口的 `70vh`
4. 列表区独立滚动
5. 底部保留安全区内边距，避免贴近手势条

### 单列卡片列表

每个比例项为整行点按卡片，保证较高命中面积。

卡片内容布局：

1. 左侧为比例缩略预览
2. 中间上方为 `比例 + 名称`
3. 中间下方为场景说明
4. 右侧为选中态标记

选中态视觉规则：

1. 主色描边
2. 浅主色背景
3. 右侧显示勾选或实心状态点

## 交互规则

### 比例状态

文生图与图生图继续分别维护各自的比例状态：

1. `textAspectRatio`
2. `imageAspectRatio`

切换模式时，读取对应模式当前已保存的比例值。抽屉选择仅修改当前模式的比例状态。

### 抽屉行为

1. 点击触发器打开抽屉
2. 点击卡片项立即更新比例
3. 更新后关闭抽屉
4. 再次打开时，当前项处于选中态并滚动可见

### 保持不变的行为

以下逻辑保持现状：

1. 输出质量选择方式
2. 生成张数选择方式
3. 质量价格计算方式
4. 比例与质量到实际 `size` 的映射
5. 文生图与图生图请求参数结构
